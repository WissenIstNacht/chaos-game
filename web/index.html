Chaos Game

An iterated function system producing a fractal.

Visualization

Description

Algorithm

The Algorithm consists of a few (infinitely) repeating steps. Given three fixed vertices v1, v2, v3 (the initial triange) and an additional, freely choosable vertex vcurr , the algorithm

  1. Chooses one of the three triangle vertices v1, v2, v3 randomly
  2. Computes the half-way point between the current point and that trianle vertex
  3. Sets that computed point as the new vcurr.

Result

The result of running the chaose game for an (infinite) amount of time is the famous Sierpinski triangle. That is, if the initial vcurr is choosen inside the initial triangle
